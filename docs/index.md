## What is AutoEvolve Company?

AutoEvolve Company is an AI Agent team that:
- Receives user requirements
- Analyzes task complexity
- Designs appropriate system architecture
- Implements code with tests
- Delivers complete software projects

## Key Features

- **Dynamic Agent Activation**: Activates only needed agents based on task complexity
- **Skill Hub**: Reusable skill system with core, OSS, and evolved skills
- **Self-Evolution**: Learns from feedback to improve performance
